serialport-tools
================

Some unix tools, that can come handy when working with serial ports.


Dependencies
------------

- [libserialport](http://sigrok.org/wiki/Libserialport) `>= 0.1.0`
- [popt](https://github.com/devzero2000/POPT) `>= 1.16`
